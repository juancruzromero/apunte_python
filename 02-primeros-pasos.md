# Primeros Pasos #

## Tipos de Datos ##
Se definen por su contenido.

```python
# String
'Hola'

# Integer
1
2
3

# Float
1,0
2,1
3,2

# Bolean
True
False
```

## Operadores ##
- Aritméticos: +,-,*,/,%,**,//.
- Comparación: <,>,=,
- Lógicos:  and, or, not.
- Asignación: =, += , -=
- Especiales: is, is not, in, not in.
